# Archive — Historical Record

This folder preserves the original artifacts behind the SGLC (Software Generation
Life Cycle) thesis as a **frozen historical record**. Files here are not maintained
and should not be edited. They are kept for provenance and to document how the idea
evolved.

This repository is a publication snapshot. Its git history begins at
the `v0.1.0` tag. Authorship of the files below predates that tag; the
self-declared dates within each document are the timeline of authorship.
